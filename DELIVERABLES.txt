BIOLOGICAL SIGNIFICANCE
================================================================================

WHY TELOTRONS NEED NON-TELOMERIC TERMINALS:

1. SPLICE SITE RECOGNITION
   - 5' donor (GT/CT) cannot function within repeats
   - 3' acceptor (AG) must be distinguished from cryptic sites
   - Clear landmark sequences required outside repeats

2. BRANCH POINT REQUIREMENT
   - Branch point = CTRAY (C-T-R-Y where R=purine, Y=pyrimidine)
   - TTAGGG cannot form: No C (cannot start CT)
   - Branch point MUST be in leader, 20-50 bp from true 3' site

3. POLYPYRIMIDINE TRACT
   - C/T-rich region upstream of 3' acceptor
   - Helps U2AF65 bind and recruit U2 snRNP
   - Distinguishes true AG from cryptic sites

4. CRYPTIC SPLICING PREVENTION
   - Each hexamer contains AG (a "trap")
   - Spliceosome could recognize first AG (truncate transcript)
   - Prevention requires strong upstream signals (in leader)

5. EVOLUTIONARY CONSERVATION
   - Symmetric leader/trailer (both ~53 bp) suggests equal constraint
   - Terminal sequences likely more conserved than array
   - Active selection to maintain splicing fidelity

TELOTRONS ARE FUNCTIONAL GENETIC ELEMENTS:
  ✓ Complex introns in genes with short ORFs
  ✓ Subject to precise splicing regulation
  ✓ Found in marine eukaryotes (niche-specific)
  ✓ Represent evolutionary innovation in splicing

CONCLUSIONS
================================================================================

1. Telotrons have CONSERVED terminal architecture:
   - 5' leaders and 3' trailers are nearly identical (53 bp each)
   - Suggests equal functional constraint
   - More conserved than telomeric arrays

2. CRYPTIC SPLICING creates structural constraints:
   - AG acceptor appears at position 2-3 of every TTAGGG
   - Each hexamer is potential truncation point
   - Complex splicing regulation required

3. Terminal sequences are FUNCTIONALLY CRITICAL:
   - NOT random spacers
   - Essential for defining 5' donor and 3' acceptor sites
   - Must contain branch point and polypyrimidine tract

4. Splicing mechanism is COMPLEX:
   - Requires proper 5' site definition (leader)
   - Requires branch point element (leader)
   - Requires polypyrimidine tract (leader/junction)
   - Requires masking of cryptic AG sites

5. Telotrons represent EVOLUTIONARY INNOVATION:
   - Unique to marine eukaryotes
   - Demonstrate complex splicing regulation
   - Provide insight into repetitive sequence splicing
   - May have functional roles in telomere biology
